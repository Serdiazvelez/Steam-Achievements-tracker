# Bombo Widgets

##Steam achievements tracker

Este proyecto es una pagina web la cual genera un link para colocar en OBS, y poder colocar un tracker de logros de algun Juego que quieras completar.

## Cómo probar localmente

1. **Ejecuta el servidor local:**
   ```bash
   node server.js
   ```

2. **Abre en el navegador:**
   ```
   http://localhost:3000
   ```

3.- **Introduce Steamid (Usuario debe ser publico)**

4.- **Seleccionar un juego y un logro.**

5.- **Se generara un link y un preview para hacer embbed en OBS.**
